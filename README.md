# FindFix — Technician Marketplace (Kenya)

A premium full-stack platform connecting customers with verified technicians.

## Features

- **Customers**: Search by service/location, post requests, chat, pay via M-Pesa.
- **Technicians**: Profile management, job acceptance, earnings tracking.
- **Admins**: Verification, user management, dispute handling, stats.

## Tech Stack

- **Frontend**: Next.js, Vanilla CSS (Glassmorphism).
- **Backend**: Node.js, Express, Socket.io, JWT.
- **Database**: PostgreSQL.
- **Payments**: M-Pesa Daraja API.

## Quick Start

1. **Backend**: `cd backend && node server.js` (after DB setup).
2. **Frontend**: `cd frontend && npm run dev`.
